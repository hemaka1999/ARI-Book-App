# ARI Book StX

ARI Book StX is a CRUD (Create, Read, Update, Delete) Laravel web application for managing a book store. It features Laravel authentication for login and registration. After logging in, users can manage the book store by adding new books, editing existing book details, and deleting books. The system also indicates when a book is out of stock.

## Introduction

ARI Book StX simplifies book store management with a user-friendly interface. It provides essential functionality, including adding, updating, and deleting books. The built-in authentication system ensures secure access for authorized users.

## Technologies and Frameworks

ARI Book StX is built using the following technologies and frameworks:

- **Backend Framework:** [Laravel](https://laravel.com/)
- **Styling Framework:** [Bootstrap](https://getbootstrap.com/)
- **Database:** MySQL
- **Languages:** CSS, JavaScript
- **Authentication:** Laravel

## Features

- User authentication with login and registration.
- Create, Read, Update, and Delete (CRUD) operations for managing books.
- Indication of book availability, showing "out of stock" when necessary.

## Installation

To run ARI Book StX on your local machine, follow these steps:

```bash
# Clone the repository
git clone https://github.com/hemaka1999/ARI-Book-App.git

# Navigate to the project directory
cd ARI-Book-App

# Install dependencies
composer install
npm install

```

## Usage

To use ARI Book StX, follow these steps:

1. Start the development server.

2. Open ARI Book StX in your web browser.

3. Log in to the application using your credentials.

4. Manage the book store by adding, editing, and deleting books.

5. Observe the "out of stock" indication as needed.
